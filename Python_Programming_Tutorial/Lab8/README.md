## 1. คลาส PhoneBattery
ให้นักศึกษาสร้างคลาส `PhoneBattery` โดยมีรายละเอียดดังนี้

1. กำหนด Private attribute `__percent` เพื่อเก็บค่าเปอร์เซ็นต์แบตเตอรี่ โดยให้มีค่าเริ่มต้นที่ `100`
2. สร้างเมธอด `use_battery(amount)` สำหรับลดปริมาณแบตเตอรี่ตามจำนวนที่ระบุ (ลดแล้วต้องไม่ต่ำกว่า `0`)
3. สร้างเมธอด `charge_battery(amount)` สำหรับเพิ่มปริมาณแบตเตอรี่ตามจำนวนที่ระบุ (เพิ่มแล้วต้องไม่เกิน `100`)
4. หากการลดหรือเพิ่มทำให้ค่าเกินขอบเขต `0-100` ให้ปรับค่าให้อยู่ในขอบเขตนั้น
5. สร้างเมธอดประเภท Getter สำหรับอ่านค่าเปอร์เซ็นต์แบตเตอรี่ปัจจุบัน (`__percent`)

จากนั้นสร้างวัตถุ (Object) จากคลาสดังกล่าว และทดสอบการเรียกใช้เมธอดต่าง ๆ พร้อมแสดงผลลัพธ์ทางหน้าจอ

## 2. Inheritance: BankAccount และ SavingsAccount
ให้นักศึกษาเขียนโปรแกรมโดยใช้หลักการ Inheritance ตามรายละเอียดต่อไปนี้

### 2.1 คลาสแม่ BankAccount
กำหนดคุณสมบัติและพฤติกรรมดังนี้

- Constructor (`__init__`)
	- รับพารามิเตอร์ 2 ตัว: `account_number` (เลขที่บัญชี) และ `balance` (ยอดเงินคงเหลือ)
	- กำหนดให้ `_balance` (Protected) มีค่าเริ่มต้นเป็น `0`

- Method `deposit(amount)`
	- รับจำนวนเงินที่ต้องการฝาก
	- ตรวจสอบว่า `amount > 0`
	- ถ้าจริง: เพิ่มยอดเงินใน `_balance` และแสดงข้อความ
		- `ฝาก {amount} บัญชี {account_number} ยอด {_balance}`
	- ถ้าเท็จ: แสดงข้อความ
		- `จำนวนเงินต้องมากกว่า 0`

- Method `withdraw(amount)`
	- รับจำนวนเงินที่ต้องการถอน
	- ตรวจสอบว่า `amount > 0` และ `amount <= _balance`
	- ถ้าจริง: หักยอดเงินจาก `_balance` และแสดงข้อความ
		- `ถอน {amount} บัญชี {account_number} ยอด {_balance}`
	- ถ้าเท็จ: แสดงข้อความ
		- `ยอดเงินไม่พอหรือจำนวนเงินไม่ถูกต้อง`

- Method `get_balance()`
	- คืนค่ายอดเงินคงเหลือปัจจุบัน

### 2.2 คลาสลูก SavingsAccount
ให้สืบทอดจาก `BankAccount` และเพิ่มรายละเอียดดังนี้

- Class Attribute
	- `interest_rate = 0.015` (ดอกเบี้ย 1.5%)

- Constructor (`__init__`)
	- รับ `account_number` และ `balance` (ค่าเริ่มต้น `0`)
	- เรียกใช้ Constructor ของคลาสแม่เพื่อกำหนดค่าเริ่มต้น

- Method `add_interest()`
	- คำนวณดอกเบี้ยจากสูตร `ดอกเบี้ย = ยอดเงินปัจจุบัน * interest_rate`
	- นำดอกเบี้ยบวกเพิ่มเข้า `_balance`
	- แสดงข้อความรูปแบบ
		- `เพิ่มดอกเบี้ย {ค่าดอกเบี้ยทศนิยม 2 ตำแหน่ง} บัญชี {account_number}`
