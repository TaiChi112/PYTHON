ให้นักศึกษาเขียนฟังก์ชัน distance1, distance2, distance3 และ perimeter ตามรายละเอียดต่อไปนี้

## 1) distance1
ฟังก์ชัน distance1(x1, y1, x2, y2)

- คืนค่าระยะห่างระหว่างจุด (x1, y1) กับ (x2, y2)
- ตัวอย่างการใช้:
	- d1 = distance1(0.0, 0, 3, 4)
	- ได้ d1 = 5.0

## 2) distance2
ฟังก์ชัน distance2(p1, p2)

- p1 และ p2 เป็นลิสต์แทนจุด โดยแต่ละลิสต์มี 2 ค่า คือ x และ y
- คืนค่าระยะห่างระหว่างจุด p1 กับ p2
- ตัวอย่างการใช้:
	- d2 = distance2([0.0, 0], [3, 4])
	- ได้ d2 = 5.0

## 3) distance3
ฟังก์ชัน distance3(c1, c2)

- c1 และ c2 แทนวงกลม 2 วง
- แต่ละวงกลมเป็นลิสต์ 3 ค่า: x, y (พิกัดจุดศูนย์กลาง) และ r (รัศมี)
- คืนค่า 2 อย่าง:
	1. ระยะห่างระหว่างจุดศูนย์กลางของ c1 กับ c2
	2. ค่า True/False ว่าวงกลมทั้งสองแตะหรือทับกันหรือไม่
- ตัวอย่างการใช้:
	- d3, overlap = distance3([0.0, 0, 1], [5, 0, 2])
	- ได้ d3 = 5.0 และ overlap = False

## 4) perimeter
ฟังก์ชัน perimeter(points)

- points เป็นลิสต์ของจุดหลายจุด โดยแต่ละจุดเป็นลิสต์ 2 ค่า [x, y]
- จุดทั้งหมดเป็นลำดับมุมของรูปหลายเหลี่ยม
- ถ้ารูปเป็น k เหลี่ยม จะมีจุด k จุด (k >= 3)
- คืนค่าความยาวรอบรูปของรูปหลายเหลี่ยมที่กำหนดโดย points

print(distance1(0, 0, 3, 4))
5.0

print(distance2([0,0], [3,4]))
5.0

a, b = distance3([0,0,1], [5,0,2]); print(a, b)
5.0 False
