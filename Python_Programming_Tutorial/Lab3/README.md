# Extra Assignment

## 1) Circle Relationship Detector
จงเขียนโปรแกรมเพื่อตรวจสอบว่าวงกลม 2 วงมีความสัมพันธ์แบบใด

- `overlap` = วงกลมทับซ้อนกัน
- `touch` = วงกลมแตะกันพอดี
- `free` = วงกลมแยกจากกัน

ข้อมูลนำเข้า:

- รับข้อมูลวงกลม 2 วง (บรรทัดละ 1 วง)
- แต่ละบรรทัดมี `x y r` (พิกัดศูนย์กลางและรัศมี)

วิธีคิด:

1. หาระยะระหว่างจุดศูนย์กลาง

$$
d = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}
$$

2. คำนวณผลรวมรัศมี $r_1 + r_2$
3. เปรียบเทียบ

- ถ้า $d = r_1 + r_2$ แสดง `touch`
- ถ้า $d < r_1 + r_2$ แสดง `overlap`
- ถ้า $d > r_1 + r_2$ แสดง `free`

## 2) Fuel Required Calculator
จงเขียนโปรแกรมคำนวณปริมาณน้ำมันที่ใช้ในการเดินทาง

ข้อมูลนำเข้า:

1. ระยะทางที่เดินทาง (กม.)
2. ประเภทรถ
	- `1` = รถยนต์ขนาดเล็ก
	- `2` = รถยนต์ขนาดกลาง
	- `3` = รถตู้
3. ลักษณะการขับขี่
	- `1` = ความเร็วไม่เกิน 80 กม./ชม.
	- `2` = ความเร็วเกิน 80 กม./ชม.

อัตราสิ้นเปลืองเฉลี่ย (กม./ลิตร):

- รถเล็ก: แบบ 1 = 12, แบบ 2 = 10
- รถกลาง: แบบ 1 = 10, แบบ 2 = 8
- รถตู้: แบบ 1 = 9, แบบ 2 = 7

สูตร:

$$
	ext{Fuel required} = \frac{\text{distance}}{\text{km per litre}}
$$

แสดงผลเป็นทศนิยม 2 ตำแหน่ง

## 3) Multiple Choice Answer Checker
จงเขียนโปรแกรมตรวจคำตอบข้อสอบปรนัย โดยมีคุณสมบัติ

1. รับสตริงเฉลยและสตริงคำตอบนักเรียนแบบต่อเนื่อง
2. ถ้าผู้ใช้พิมพ์ `exit` (ไม่สนตัวพิมพ์เล็ก-ใหญ่) ในช่องเฉลย ให้จบโปรแกรมและแสดง
	- `Exiting program. Goodbye!`
3. เปรียบเทียบคำตอบแบบไม่สนตัวพิมพ์เล็ก-ใหญ่
4. ถ้าจำนวนข้อของเฉลยและคำตอบไม่เท่ากัน ให้แสดง
	- `Incomplete answer`
5. ถ้าความยาวเท่ากัน ให้แสดง
	- `Number of correct answers: [จำนวนข้อที่ถูก]`

## 4) Thai ID Checksum Calculator
จงเขียนโปรแกรมคำนวณ checksum ของเลขประจำตัวประชาชน 13 หลัก

ขั้นตอนคำนวณ:

1. นำเลข 12 หลักแรกคูณกับค่าน้ำหนักจาก 13 ถึง 2 (ซ้ายไปขวา)
2. นำผลคูณทั้งหมดมาบวกกัน
3. นำผลรวมหาร 11 และหาเศษ
4. นำ `11 - เศษ`
5. เอาหลักหน่วยของผลลัพธ์ข้อ 4 เป็น checksum digit

ตัวอย่าง:

- Input: `3795357249057`
- Calculated checksum digit: `7`
